Timestamp Microservice

Here's how it goes,

Submit a date like the following:

You can submit a natural date at the end of the app URL like so: https://better-latency.glitch.me/December 15, 2015 (click link or copy paste into your browser address bar and hit enter)

Which will return the date you entered and the unix timestamp of the date you entered in a JSON string

You can submit a unix timestamp date at the end of the app URL like so: https://better-latency.glitch.me/1450137600 (click link or copy paste into your browser address bar and hit enter)

Which will return the unix timestamp you entered and the the equivalent natural date in a JSON string
